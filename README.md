# Embedded Software Programming

BeagleBoard에 U-boot 부트로더로 부팅하고 TeraTerm으로 Linux 개발서버에 로그인하여 프로그램한 뒤 USB Serial Port와 연결된 보드에 kermit하고 특정 메모리 주소(0x82000000)에 다이렉트로 go 명령을 내려 실행한 값들을 출력하고 확인합니다.
